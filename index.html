Ayesha's Reflection

For A2, we were told that the pdf seemed rushed (I made it last time), so this time I decided to start working on the pdf report before-hand so that I have the time to format it and match the font to the website, and keep the font sizes and colours consistent throughout the document. From past experiences, I learned that starting work on the assignment on time and making sure to read the specifications and go through it thoroughly is very important, so I made sure to that and hence, working on this assignment was a little less hectic. This assignment went a lot smoother for me compared to last time, however, I cannot be sure that it was the same for the other members. I can only hope that it was. What could and should have improved is the group communication. I was hoping that it would get better but instead it ended up becoming worse. It is understandable that some members had some problems and constraints due to which communication was a problem, but if I’m being completely honest, not being able to get your message along, or not getting a response from someone can make working in a group very tough. Despite the trouble that our group faced, we were able to work it out and we managed to finish the assignment! We did our best and I hope we get good marks this time as well!
